Descargar Repositorio Localmente

  1. Entrar a GitHub
  2. Iniciar sesión
  3. Copiar ruta del repositorio remoto
  4. Abrir git-bash
  5. Movernos al escritorio con el uso del comando "cd ~/Desktop"
  6. clonar el repositorio "gitclone" y pegar la ruta
  7. Movernos al repositorio local (ruta) con el uso del comando "cd DesarrolloWeb/"
  8. Verificar que nos encontemos en main
